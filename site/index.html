Install URL

https://gkfirst8.github.io/eclipse-xslt-3-support/

Paste this into Help > Install New Software... inside Eclipse.

Install In Eclipse

Five quick steps

  1. Open Eclipse.
  2. Choose Help > Install New Software....
  3. Click Add....
  4. Enter https://gkfirst8.github.io/eclipse-xslt-3-support/ as the location.
  5. Select XSLT 3 Support and complete the install wizard.

If the repository does not appear immediately, wait a moment for Eclipse to load the p2 metadata from this same URL.

  • Outline for common top-level declarations
  • Content assist for common XSLT elements

Troubleshooting

If Eclipse does not list the feature

  • Confirm you used the root URL exactly, including the trailing slash.
  • Retry after a short delay if a new release was just published.
  • Open the raw p2.index link in a browser to confirm the site is reachable.
  • Check Eclipse installation details if your target platform blocks unsigned additions.
